# LAN Messenger (asyncio + websockets)

Простой децентрализованный LAN-мессенджер реального времени.

## Запуск сервера

```bash
pip install -r requirements.txt
python server.py
```

По умолчанию:

- HTTP-сервер: `http://<IP_СЕРВЕРА>:8000/` — отдаёт `index.html`
- WebSocket-сервер: `ws://<IP_СЕРВЕРА>:8765/ws`

IP-адрес сервера — это IP машины в локальной сети, на которой вы запускаете `server.py`.

## Подключение клиента

1. Убедитесь, что сервер запущен.
2. На другом устройстве в той же локальной сети откройте браузер и перейдите по адресу:
   - `http://<IP_СЕРВЕРА>:8000/`
3. Введите ник и начните общение.

## Ограничения

- Нет шифрования (HTTP + WebSocket, без TLS).
- Нет регистрации и постоянных аккаунтов.
- История сообщений хранится только в памяти сервера и теряется при перезапуске.

## iOS-клиент

В проект добавлен нативный SwiftUI-клиент в папке `ios/LocalChatIOS`.

Запуск:

1. Перейдите в `ios/LocalChatIOS`
2. Сгенерируйте Xcode-проект через XcodeGen:
   - `brew install xcodegen`
   - `xcodegen generate`
3. Откройте `LocalChatIOS.xcodeproj` и запустите приложение
