# LocalChat iOS

Нативный SwiftUI-клиент для вашего LAN WebSocket чата.

## Что реализовано

- Подключение к `ws://<server-ip>:8765/ws`
- Регистрация пользователя (`type: register`)
- Прием системных сообщений, истории, списка пользователей и новых чатов
- Отправка сообщений (`type: chat`)

## Быстрый старт (через XcodeGen)

1. Установите XcodeGen:
   ```bash
   brew install xcodegen
   ```
2. В этой папке выполните:
   ```bash
   xcodegen generate
   ```
3. Откройте `LocalChatIOS.xcodeproj` в Xcode.
4. Запустите приложение на симуляторе или iPhone.

## Важные замечания

- Укажите IP сервера в поле адреса в приложении, например `ws://192.168.1.10:8765/ws`.
- iOS-симулятор должен видеть вашу локальную сеть.
- Сервер в текущем проекте должен быть запущен (`python server.py`).
